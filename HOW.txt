 Сюжет:
трехэтапный CTF с возрастающей сложностью. Каждый этап потребует разгадывания головоломки, а найденное слово нужно будет подставить в URL , Admin : , Console: , чтобы перейти дальше.

В 2 , 3 этапе есть свой FLAG{}

Этап 1: Главная страница

Страница предлагает найти ключ и подставить его в URL, например: /?key=flag, чтобы перейти ко второму этапу.
Также есть скрытая подсказка в JavaScript (файл hint.js), которая может помочь найти ключ.
Этап 2: SQL-инъекция

1.Encypt
2.НАйти логин и пароль 

Этап 3: Тайный API

Здесь используется подделка заголовков через JavaScript (auth.js), чтобы получить доступ к API.
API использует заголовок X-Secret-Key, который можно подделать с помощью JavaScript.
После подделки запроса через JavaScript будет выведен финальный флаг.
